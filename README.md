## Feature

- 즐겨찾기 / 그룹 (팔로우 채널 중 우선순위 올리기)
- 생방송 실시간 미리보기
- 레이아웃 커스텀 구성하기
- 통나무 자동 수집
- 방송 시작 알림
- 방송 종료 알림
- 오디오 압축 (오디오 평준화 및 고음역 보호)
- 채팅 저장소 (자주 사용하는 채팅 저장하고 쉽게 복사하기)
- 특정 유저 채팅 고정
- 비도오 화면 필터
- 생방송 지연 시간 표시
- 생방송 빨리감기
- 시청자수 브라우저탭 표기
- 채팅 색상 모드 (커스텀 색상, 채팅 색상)
- 채팅 크기 조절
- 치즈 채팅 끄기
- 블라인드 채팅 끄기
- 구독 채팅 끄기
- 생방송 시작시 자동 새로고침
- 차단한 방송 완전 제거
- 팔로우 채널 자동 새로고침
- 화면 보호기
- 채팅 뱃지 제거거
- 단축키 기능
- 자동 넓은 화면
- 새 탭으로 열기
