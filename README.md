## Feature

1. 즐겨찾기
2. 생방송 실시간 미리보기
3. 채팅 색상 모드 (커스텀 색상, 채팅 색상)
4. 채팅 크기 조절
5. 치즈 채팅 끄기
6. 블라인드 채팅 끄기
7. 구독 채팅 끄기
8. 즐겨찾기 (팔로우 채널 중 우선순위 올리기)
9. 생방송 빨리감기
10. 오디오 압축 (오디오 평준화 및 고음역 보호)
11. 채팅 저장소 (자주 사용하는 채팅 저장하고 쉽게 복사하기)
12. 생방송 시작시 자동 새로고침
13. 차단한 방송 완전 제거
14. 팔로우 채널 자동 새로고침
15. 유저 채팅 고정
16. 화면 보호기
17. 채팅 뱃지 제거거
18. 단축키 기능
19. 자동 넓은 화면
